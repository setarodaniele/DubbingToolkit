DESCRIZIONE
Dubbing Toolkit è un sistema per la gestione del workflow di doppiaggio:
- estrazione audio da video
- trascrizione automatica
- traduzione
- generazione contenuti doppiati

AVVIO RAPIDO
1. Esegui StartDubbing.bat
2. Inserisci i file in:
   - Video_Input → video sorgente
   - Audio_Input → input audio (se necessario)
