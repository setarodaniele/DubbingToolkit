# Часто задаваемые вопросы

---

## Запуск

**Ошибка политики выполнения PowerShell**

Если при запуске появляется сообщение об ошибке политики выполнения скриптов:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

Выполните эту команду в PowerShell, затем снова запустите `StartDubbing.bat`.

---

**Виртуальное окружение повреждено**

Если возникают ошибки импорта или запуск не удаётся:

1. Удалите папку `venv/`
2. Запустите `StartDubbing.bat` — окружение будет создано заново автоматически

---

**В интерфейсе отображается `[MISSING: key]`**

Это означает, что в файле локализации отсутствует ключ для текущего языка интерфейса. Ошибка некритическая и не влияет на работу инструмента. Сообщите об ошибке разработчику, указав язык интерфейса и текст `[MISSING: ...]`.

---

## Учётные данные

**Azure: учётные данные недействительны**

Проверьте файл `credentials/azure_speech_credentials.json`:

```json
{
  "subscription": "ваш_ключ_azure",
  "region": "ваш_регион"
}
```

- `subscription` — 32-символьный ключ API из раздела «Ключи и конечная точка» в Azure Portal
- `region` — должен точно совпадать с регионом ресурса (например, `westeurope`, `eastus`)

Подробнее: [credenziali_azure.md](credenziali_azure.md)

---

**Google: учётные данные недействительны**

Файл `credentials/google_speech_credentials.json` должен содержать полный JSON сервисного аккаунта GCP.

- Убедитесь, что для проекта включён API **Cloud Text-to-Speech**
- Файл должен содержать все обязательные поля, включая `private_key` в формате PEM

Подробнее: [credenziali_google.md](credenziali_google.md)

## Транскрипция

**Распознан неправильный язык**

Если Whisper определил язык аудио неверно, укажите язык вручную через меню настроек (`Transcript_Audio_Spoken_Lang`).

---

**Транскрипция работает слишком медленно**

Выберите более лёгкую модель Whisper в меню настроек:

| Модель | Скорость | Качество |
|--------|----------|----------|
| `tiny` | Очень быстро | Базовое |
| `base` | Быстро | Удовлетворительное |
| `small` | Среднее | Хорошее |
| `medium` | Медленно | Высокое |
| `large` | Очень медленно | Максимальное |

---

**Ошибка нехватки памяти при транскрипции**

Разбейте аудиофайл на более короткие сегменты и обработайте их по отдельности. Поблочная обработка запланирована в будущих версиях.

---

## Перевод

**Языковая пара недоступна**

Helsinki-NLP поддерживает ограниченный набор языковых пар. Прямой перевод между всеми языками может быть недоступен. Трансферный перевод через английский язык запланирован на будущее.

---

**Перевод содержит ошибки или звучит неестественно**

Helsinki-NLP — это локальные модели. На идиомах и технических терминах они могут уступать облачным сервисам. Ручная правка переведённых SRT-файлов перед синтезом речи всегда рекомендуется. Постобработка текста запланирована на будущее.

---

## Синтез речи (TTS)

**Неестественные паузы в синтезированном аудио**

Используйте нейронные голоса: **Azure Neural** или **Google WaveNet/Neural2**. Они обеспечивают более плавную и естественную интонацию по сравнению со стандартными голосами.

---

**Синтезированное аудио тихое или содержит шум**

Проверьте папку `Translated/` — возможно, некоторые сегменты SRT пустые или содержат некорректный текст. Убедитесь, что файл перевода был проверен перед запуском TTS.

---

**Статистика использования не обновляется**

Если счётчик использования TTS (`Billing/consumo_tts.json`) ведёт себя некорректно:

1. Создайте резервную копию файла `Billing/consumo_tts.json`
2. Удалите оригинал
3. Запустите TTS заново — файл будет создан с нуля

## Файлы и папки

**Где находятся выходные файлы?**

| Этап | Папка |
|------|-------|
| Извлечённое аудио | `Audio_Extracted/<timestamp>_<имя_файла>/` |
| Транскрипция (SRT) | `Transcripts/<timestamp>_<имя_файла>/` |
| Перевод (SRT) | `Translated/<timestamp>_<имя_файла>/` |
| Синтез речи | `Dubbed/<ПРОВАЙДЕР>/<timestamp>_<имя_файла>/` |

Каждая подпапка содержит файл `_info.txt` с подробными метаданными об обработке.

---

**Я переместил папку проекта, теперь ничего не работает**

Удалите папку `venv/` и запустите `StartDubbing.bat` из нового расположения. Виртуальное окружение будет создано заново с правильными путями.

## Сборка и распространение

**Почему в установочном пакете нет учётных данных?**

Это сделано намеренно из соображений безопасности. Учётные данные персональны и привязаны к биллингу. После установки добавьте их вручную в папку `credentials/`, используя шаблоны `.template.json`.

## Общие вопросы

**Можно ли работать без интернета?**

Частично. Извлечение аудио, транскрипция (Whisper работает локально) и перевод (Helsinki-NLP работает локально) доступны без интернета. TTS (Azure и Google) **требует** подключение к интернету. Загрузка моделей перевода при первом использовании также требует интернет.

---

**Как добавить новый язык интерфейса?**

1. Создайте файл `locale/Active/<код_языка>.json` по образцу существующего файла
2. Добавьте запись в `locale/System/languages.json`
3. Новый язык появится в меню выбора языка интерфейса

---

**Доступна ли пакетная обработка нескольких файлов?**

Пока нет. Система обрабатывает по одному файлу за раз. Пакетная обработка запланирована на будущее.
