# DubbingToolkit

> ⚠️ **Предупреждение**: Эта документация была переведена автоматически и может содержать ошибки или неточности. Для подробного ознакомления обратитесь к версии на [английском языке](../en/README.md).

DubbingToolkit — это гибридная система озвучивания на Python + PowerShell, предназначенная для автоматизации озвучивания обучающих видео на Windows. Инструмент объединяет извлечение аудио (ffmpeg), транскрипцию речи (OpenAI Whisper → SRT), нейронный машинный перевод (Helsinki-NLP, локальные модели) и синтез речи (Azure или Google, посегментно) в единый CLI-рабочий процесс.

## Функциональность

DubbingToolkit автоматизирует полный цикл озвучивания в четыре этапа:

1. **Извлечение аудио** — ffmpeg извлекает аудиодорожки из видеофайлов
2. **Транскрипция** — OpenAI Whisper преобразует аудио в текст в формате SRT
3. **Перевод** — Helsinki-NLP (локальные модели MarianMT) переводит субтитры
4. **Синтез речи (TTS)** — Azure Cognitive Services или Google Cloud TTS генерируют озвученный аудиофайл

## Точка входа

```bat
StartDubbing.bat
```

Двойной клик по `StartDubbing.bat` запускает всю цепочку: активирует среду Python, проверяет зависимости и открывает главное меню.

## Провайдеры TTS

| Провайдер | Описание |
|-----------|----------|
| **Azure Cognitive Services Speech** | Нейронные голоса Microsoft, широкий выбор языков и голосов |
| **Google Cloud Text-to-Speech** | Голоса WaveNet и Neural2 от Google, высокое качество |

Для работы TTS требуются действительные учётные данные. Подробнее см. [credenziali_azure.md](credenziali_azure.md) и [credenziali_google.md](credenziali_google.md).
